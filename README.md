# Trading Bots Dashboard


## Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/erkinchik/trading-bot.git
```

### 2. Установите зависимости

```bash 
npm install
```

### 3. Запустите приложение

```bash 
npm start
```
